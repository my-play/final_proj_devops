Steps to Deploy
	1.	Add all files to the repository:

    git add .
    git commit -m "Add TerraformCD workflow"
    git push origin main


	2.	Configure GitHub Secrets:
	•	AWS_ACCESS_KEY_ID
	•	AWS_SECRET_ACCESS_KEY
	3.	Trigger the workflow by pushing changes to the main branch.
